Esse é um projeto foi inspirado num exercício de treinar a desenvolvimento de raciocínio lógico, o Jogo de Einstein.  
Fiz ele com o intuito de treinar minha lógica de programação no C# (.Net Framework 4.8) durante meu período de formação no técnico em informática.   
    
Com esse projeto pude apredrer e aprimorar meu conhecimento em didaticas como:  
  -> Programação Orientada Objetos. 
  -> Lógica de programação  
  -> Estruturas condicionais    

Para vocês poderem jogar e testar seu raciocínio, basta clonar esse repositório e executar na IDE que você ultiliza para trabalhar em C# (Visual Studio por exemplo).
